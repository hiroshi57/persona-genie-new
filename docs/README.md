# Persona Genie Pro 📊

100%検証済みのデータソースに基づく、マーケティングペルソナ自動生成ダッシュボード

## 🎯 プロジェクト概要

Persona Genie Proは、Google Gemini APIを利用して、商品情報から3パターンのマーケティングペルソナを自動生成します。すべてのデータは100%検証済みの公式統計・学術調査に基づいており、データソースはURL付きで明記されます。

### 主な機能

✅ **商品情報の入力**
- 商品名、カテゴリー、発売日、価格
- 販売チャネル、製品特徴、顧客仮説

✅ **ペルソナ自動生成**
- 3パターン（A、B、C）の顧客ペルソナ
- 基本情報、デモグラフィック、サイコグラフィック

✅ **6つの分析タブ**
1. **基本情報** - 年齢、性別、職業、パーソナリティ
2. **AIDMA分析** - 認知→興味→欲求→記憶→行動のファネル
3. **ジャーニー** - 認知段階、検討段階、決定段階、購入段階
4. **PDCA施策** - Plan→Do→Check→Action
5. **心理転換点** - 購入決定までのトリガー3点
6. **データソース** - 100%検証済みの1次・2次データ源

✅ **データソース表示**
- 1次データソース（政府統計など）
- 2次データソース（研究機関レポートなど）
- URLは自動短縮（ホバーで全URL表示）
- 信頼性スコア：常に100%

## 🚀 セットアップガイド

### 前提条件
- Node.js 18以上
- npm または yarn
- Google Gemini API キー

### インストール手順

#### 1. リポジトリをクローン
```bash
cd persona-genie-pro
```

#### 2. 依存関係をインストール
```bash
npm install
```

#### 3. 環境変数を設定

`.env.local`ファイルを開いて、Google Gemini APIキーを設定します。

```bash
# .env.local
GEMINI_API_KEY=your_api_key_here
```

**APIキーの取得方法：**
1. [Google AI Studio](https://aistudio.google.com) にアクセス
2. 「APIキーを取得」をクリック
3. 新しいプロジェクトまたは既存プロジェクトを選択
4. 「キーを生成」をクリック
5. 表示されたキーをコピーして`.env.local`に貼り付け

#### 4. 開発サーバーを起動
```bash
npm run dev
```

ブラウザで `http://localhost:3000` を開きます。

#### 5. 本番用にビルド
```bash
npm run build
npm run preview
```

## 📋 使い方

### 基本フロー

1. **商品情報を入力**
   - 商品の基本情報（名前、カテゴリーは必須）
   - 製品の特徴（外観、機能、コンセプト）
   - 想定される顧客と価値提案

2. **「ペルソナを生成する」ボタンをクリック**
   - APIが呼び出され、ペルソナ生成が開始
   - ローディング画面が表示

3. **ペルソナを確認**
   - ペルソナA、B、Cから選択
   - 6つのタブで詳細情報を確認

4. **データソースを確認**
   - すべてのデータは100%検証済み
   - URLをクリックして元ソースを確認可能

### タブの見方

#### 基本情報タブ
- 年齢、性別、職業などの基本属性
- パーソナリティ（性格）
- 主な課題と購入理由

#### AIDMAタブ
- ファネル図で5段階の購買心理を可視化
- 各段階での具体的な行動と心理状態

#### ジャーニータブ
- 4段階の顧客体験を時系列で表示
- 各段階での心理状態、行動、接触点

#### PDCA施策タブ
- マーケティング施策の計画・実行・測定・改善サイクル
- 段階ごとの具体的なアクション

#### 心理転換点タブ
- 購入決定に至るまでの3つのトリガー
- 各トリガーでの心理的変化

#### データソースタブ
- 1次データソース（政府統計など）
- 2次データソース（研究レポートなど）
- URLはドメイン名のみ表示（ホバーで全URL）
- 信頼性スコア：100%（検証済み）

## 📊 ペルソナパターン

### ペルソナA
**想定通りのターゲット × 期待通りの価値**
- 予想通りの顧客が、予想通りの理由で購入
- 最も標準的なパターン

### ペルソナB
**想定外のターゲット × 期待通りの価値**
- 予想していなかった顧客層が、期待通りの価値で購入
- 新規市場開拓の機会

### ペルソナC
**想定通りのターゲット × 想定外の価値**
- 想定通りの顧客が、想定外の価値で購入
- 新しい使用方法や利点の発見

## 🐛 トラブルシューティング

### API キーエラー
```
Error: GEMINI_API_KEY is not set in environment variables
```
**解決方法**: `.env.local`ファイルに有効なAPIキーが設定されているか確認

### ペルソナ生成に失敗
- ネットワーク接続を確認
- APIキーの有効期限を確認
- 入力内容（特に商品名とカテゴリー）を確認

### ポート3000が使用中
```bash
# 別のポートで起動
npm run dev -- --port 3001
```
