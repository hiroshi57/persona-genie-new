## 🆕 新規追加機能

### 5Big診断（Big Five Personality Model）

```
心理学の標準モデル：5つのパーソナリティ特性

① 開放性 (Openness)
   └─ 新しい経験への好奇心度

② 誠実性 (Conscientiousness)
   └─ 計画性・責任感の強さ

③ 外向性 (Extraversion)
   └─ 社交性・活動性の程度

④ 協調性 (Agreeableness)
   └─ 他者との協力度・思いやり

⑤ 神経症傾向 (Neuroticism)
   └─ 不安感・ストレス反応度
```

### 各スコア（0-100）の購買行動への影響

例：開放性が高い（70-100）
→ 新製品への関心が高い
→ トレンド志向の購買行動
→ マーケティング: イノベーション強調
