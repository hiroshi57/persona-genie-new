# PersonaZen 🧠

> 10万人シミュレーション × AI詳細ペルソナ生成ダッシュボード

ターゲット顧客を大規模な仮想集団としてシミュレーションし、
人間科学の4要素・Big5性格分析・「動かない理由」分析・AI生成の詳細ペルソナで
「誰に・なぜ刺さるか」を可視化するツールです。

## 🎯 どんなとき使う？

- 新商品・新サービスのターゲット設計
- マーケティング施策が刺さる層の検証
- 広告クリエイティブのペルソナ別最適化
- 事業計画・投資判断のための顧客理解

## ✅ 何に役立つ？

- 「誰に売るか」を大規模分布で定量的に把握
- 人間科学4要素 × Big5 × シミュレーションで購買行動を予測
- 「動かない理由（非購買要因）」まで分析し、施策の打ち手が見える
- AIが各クラスタの詳細ペルソナ（属性・価値観・購買動機）を自動生成

## 🚀 開発

```bash
npm install
npm run dev      # 開発サーバー
npm run build    # 本番ビルド
npm run preview  # ビルド確認
```

環境変数は `.env.example` を参照（AI APIキー等）。
