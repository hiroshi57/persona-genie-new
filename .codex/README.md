# Persona Genie Pro v2

**1000人シミュレーション × AI詳細ペルソナ生成**

マーケティング向けの高度なペルソナ分析ツール。Box-Muller変換による正規分布、Big5性格診断、4象限分析、Claude APIを統合した企業向けダッシュボード。

## 📋 仕様

### 技術スタック
- **フロントエンド:** React 19 + TypeScript
- **ビルド:** Vite 5
- **スタイリング:** Tailwind CSS
- **UI コンポーネント:** lucide-react
- **AI API:** Claude Opus 4

### 機能

#### 1️⃣ **入力フェーズ**
- 商品情報入力（名前、カテゴリー、価格、販売チャネル等）
- 製品特徴の詳細入力（外観、機能、コンセプト）
- 顧客仮説と価値提案の定義

#### 2️⃣ **シミュレーションフェーズ**
- **1000人生成** - Box-Muller変換による正規分布
- **属性データ** - 年齢、性別、職業、収入、地域、Big5性格
- **4象限分類** - 認知度×購買意欲

#### 3️⃣ **パラメータ調整**
- 価格調整（-50% ～ +50%）
- 販売場所拡大（0% ～ 100%）
- ターゲット年齢範囲
- 所得水準
- 地域多様性

#### 4️⃣ **AI詳細ペルソナ生成**
- 3つの代表ペルソナ抽出（各象限から1人）
- **6シート統合構成**：
  1. 基本情報
  2. 心理プロファイル（Big5診断含む）
  3. 購買行動
  4. 4象限ポジション
  5. カスタマージャーニー（4段階）
  6. AI要約と推奨 + Big5解釈

#### 5️⃣ **詳細分析**
- AIDMA分析（認知→興味→欲求→記憶→行動）
- PDCA施策
- 心理転換点（複数トリガー）
- 100%検証済みデータソース付き

## 🚀 セットアップ

### 前提条件
- Node.js 18+
- npm または yarn

### インストール

```bash
cd persona-genie-pro-v2-complete

# 依存関係インストール
npm install

# API キー設定
# .env.local を編集してClaude API キーを設定
# https://console.anthropic.com から取得
```

### 開発サーバー起動

```bash
npm run dev
```

ブラウザで `http://localhost:3000` にアクセス

### プロダクションビルド

```bash
npm run build

# ビルド結果のプレビュー
npm run preview
```

## 📊 使用方法

### Step 1: 商品情報入力
```
- 商品名: 例「ワイヤレスイヤフォン」
- カテゴリー: 例「電子機器」
- 価格: 例「15000」
- 販売チャネル: 例「EC、家電量販店」
- 製品特徴を詳細入力
```

### Step 2: シミュレーション実行
「1000人シミュレーションを開始」ボタンをクリック
→ 1000人の個人属性が生成される

### Step 3: スライダーで最適化
左側のスライダーで以下を調整：
- **価格調整** - 価格感度の影響を可視化
- **販売場所拡大** - 流通拡大の効果
- **ターゲット年齢** - 年齢層の最適化
- **所得水準** - 購買力の考慮
- **地域多様性** - 地域別戦略

右側の 4象限グラフと統計情報がリアルタイム更新

### Step 4: ペルソナ生成
「3つのAI詳細ペルソナを生成」ボタンをクリック
→ Claude APIで詳細なペルソナ情報を自動生成

### Step 5: 詳細分析
各タブでペルソナの詳細情報を確認：
- **基本情報** - 人口統計情報
- **心理プロファイル** - Big5性格診断
- **購買行動** - 購買パターン分析
- **4象限ポジション** - マーケット上の位置付け
- **カスタマージャーニー** - 購買プロセスの詳細化
- **AI要約&Big5** - AIによる深層洞察

## 📈 4象限分類

| 象限 | 認知度 | 購買意欲 | 特性 | 戦略 |
|------|-------|--------|------|------|
| **Q1** | 高 | 高 | 確実顧客 | 即時購入促進 |
| **Q2** | 低 | 高 | 啓発型 | 認知施策重視 |
| **Q3** | 低 | 低 | 無関心層 | 中期施策 |
| **Q4** | 高 | 低 | 転換型 | 説得・価値提案 |

## 🧠 Big5性格診断

ペルソナの心理プロファイルを5つの因子で分析：

1. **開放性 (Openness)** - 新しい経験への開放性
2. **誠実性 (Conscientiousness)** - 計画性と責任感
3. **外向性 (Extraversion)** - 社交性と活動性
4. **調和性 (Agreeableness)** - 協調性と共感性
5. **神経症傾向 (Neuroticism)** - 感情的安定性

## 🔐 API キー取得

### Claude API キー
1. https://console.anthropic.com にアクセス
2. 「Create API Key」をクリック
3. キーを生成
4. `.env.local` に貼り付け：
```
VITE_ANTHROPIC_API_KEY=sk-ant-...
```
